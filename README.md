## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/suma1282/FUTURE_FS_01.git
```

Navigate to project folder:

```bash
cd FUTURE_FS_01
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
